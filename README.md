# Stomatološka ordinacija Hercegovina

This repository contains the seminar project **“Stomatološka ordinacija Hercegovina”**, a web-based application that simulates the management and online presence of a dental clinic.

The project is developed for academic purposes and includes:
- Presentation of dental services
- Patient information management
- Appointment scheduling (conceptual or implemented)
- Modern web application structure and best practices

**Technologies:**
- Backend: **C# (.NET), Docker, RabbitMQ, SQL**  
- Frontend: **Flutter**
  
> Seminar project – educational use only.

**Repository structure:**
- `app/` Flutter application (mobile + desktop)
- `backend/` .NET API, services, and Docker files
- `docs/` command references
- `scripts/` utility commands
